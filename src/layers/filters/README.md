# Layer Filters

本模块实现了作用于图层的各种图像滤镜（Image Filters）。滤镜通常会对图层的像素数据进行后处理，生成新的视觉效果。

## 核心接口

*   **LayerFilter**: 所有滤镜的基类。定义了滤镜的参数接口和处理逻辑。通常会生成一个 `ImageFilter` 对象传递给底层的 GPU 渲染引擎。

## 内置滤镜

*   **BlendFilter**: 混合滤镜。允许图层与其背景进行特定的混合模式（Blend Mode）合成。
*   **BlurFilter**: 高斯模糊滤镜。对图层内容进行模糊处理，支持设置模糊半径。
*   **ColorMatrixFilter**: 颜色矩阵滤镜。通过 4x5 或 5x5 矩阵变换图层的颜色（如调整饱和度、对比度、色相）。
*   **DropShadowFilter**: 投影滤镜。为图层添加外部阴影效果，支持颜色、模糊、偏移等参数。
*   **InnerShadowFilter**: 内阴影滤镜。在图层内部添加阴影效果，营造凹陷感。

## 使用方式

滤镜通常作为 `Layer` 的属性存在。渲染时，渲染管线会检查图层是否挂载了滤镜，如果有，则会先将图层内容绘制到临时缓冲区，应用滤镜效果后，再绘制到目标表面。
